Investment Platform

A full-stack investment platform with user authentication and financial features

About This Project

This is a complete investment platform with the following features:

  • User Authentication (Registration & Login)
  • Product Plans with Daily Income
  • Withdrawal System
  • Recharge System
  • Referral Sharing
  • Admin Panel
  • Mobile-First Responsive Design

Technology Stack

  • Frontend: React.js (Hosted on Netlify)
  • Backend: Node.js + Express.js (Hosted on Render)
  • Database: Supabase (PostgreSQL)
  • Authentication: JWT (JSON Web Tokens)
